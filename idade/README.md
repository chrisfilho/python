EP 3 - Idade
=================

Programa para descobrir o ano de nascimento de uma pessoa a partir da idade.

Instruções
----------
Crie um arquivo com o nome "script.py" na pasta idade.

Especificação
-----------
- Permitir a entrada da idade do usuário.
- Mostar o ano de nascimento atual

Exemplo

$ python script.py
Digite sua idade:
11 
Você nasceu em **2003**
